**BidaTask** is a community-driven task service platform designed to connect people who need help with those ready to lend a hand.
Built around the spirit of bayanihan, BidaTask allows users to post errands, accept nearby tasks, and earn through reliable peer-to-peer assistance — all within one seamless app.

Unlike traditional role-based service apps, every user in BidaTask can be both a Requester and a Tasker, promoting flexibility, fairness, and genuine collaboration. The platform fosters local connections through location-based discovery, secure wallet integration, and user reputation badges that highlight skill, reliability, and community trust.
